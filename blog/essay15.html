阿贝云搭建上网服务器 ​

​

网址：https://www.abeiyun.com/

大家好，今天给大家带来一个保号免费上网教程！

也许，有人看过类似的视频。但是我的方法和别人不同，别人都是直接提供保姆服务。而我这个呢，会让大家学会自己搭上网服务器，达到免费上网。（温馨提示：本教程只是为那些想折腾又不会折腾的人使用，大佬勿喷！）

首先，你得有一个处在公网环境的服务器，这里我选的阿贝云。服务器记得要linux系统，因为本次的主角就是Linux架构的shell脚本。

这里教一下阿贝云服务器的使用方法：

首先注册一个阿贝云的帐号，并开启一台云服务器，记住，是云服务器。（有免费的和花钱的，这里我已免费的服务器做演示）

开启后，点击“管理面板”，第一次会让你装系统。点击”重装系统“，按提示操作即可。系统安装完成后，通过ssh连接服务器。

1、进入服务器后，解锁root用户。输入以下代码：

shell
sudo passwd root
1

2、输入你的密码 su root

shell
su root
1

3、输入你刚才输入的密码

4、进入root目录，创建文件夹为“pptp-VPN”

shell
cd /root
mkdir pptp-VPN
1
2

5、在文件夹中放置脚本，这里我们通过vim在服务器中创建这个脚本。

shell
vim start.sh
1

6、在电脑复制脚本中的代码，在ssh中右键即可粘贴带服务器中

shell
然后按【ESC]输入 :wq
1

7、运行脚本就大功告成了。

shell
sh ./stat.sh
1

记得要的等待片刻，等出现账号密码时。服务器完成。

8、手机打开VPN或虚拟专用网络，点击添加将ssh中ip地址，用户名密码填入进去保存即可。连接时开启VPN就好。
