vitepress搭建静态博客 ​

1、创建项目（必须） ​

项目名字随意

text
mkdir vitepress-starter && cd vitepress-starter
1

2、初始化项目为npm项目（必须） ​

自动创建pakeage.json文件

text
npm init -y
1

3、安装vitepress和vue依赖（必须） ​

生成node_modules目录、package-lock.json

text
npm add --dev vitepress vue
1

4、创建页面（必须） ​

页面必须在docs目录下

text
mkdir docs
1

5、配置vitepress ​

创建docs/.vitepress/config.js

javascript
import {defineConfig} from 'vitepress'
export default defineConfig({
	// 网站语言en-US，zh-CN
	lang: 'en-US',
	title: '网站标题',
	// 网站描述
	description: 'Vite & Vue powered static site generator.',
	// 开启主题切换（打开黑白天按钮）
	appearance: true,
	// 设置网站根路径
	base: '/root/',
	// 设置为true时，md文档中的link转化成a标签，会将a标签href属性设置真实跳转地址；设置为false时，href属性为空
	ignoreDeadLinks: true,
	// 显示文档最后更新时间
	lastUpdated: true,
	// 网站副标题，内容为：title属性值|titleTemplate属性值
	titleTemplate: '博客'
	// 设置markdown配置
	markdown: {
		// markdown风格（不写也可以，需要vitepress中有该风格的代码）
		theme: 'material-palenight',
		// 显示markdown行号
		lineNumbers: true
	},
	// 设置导航栏和侧边栏
	themeConfig: {
    // 设置logo（位置在docs/public/logo.svg）
    logo: '/logo.svg',
    nav: [
		// 设置导航栏
		{text: '首页',link: '/'},// 链接到docs/index.md
		{text: '关于',link: '/about/'},// 链接到docs/about/index.md
		{text: '博客',link: '/blog/blog'},// 链接到docs/blog/blog.md（没有最后的/）
    ],
	// 设置公用的侧边栏（侧边栏和导航栏没有关联）
	sidebar: [
		{
			text: '侧边栏一级',
			items: [
				{ text: '侧边栏二级', link: '/blog/blog1' },// 链接到docs/blog/blog1.md
				{ text: '侧边栏二级', link: '/blog/blog2' }// 链接到docs/blog/blog2.md
			]
		}
	],
	// 或者（侧边栏和导航栏联动）
	sidebar: {
		"/blog/": [
			{
				text: '侧边栏一级',
				collapsible: true,//显示折叠（折叠的+号）
				collapsed: true,//默认不展开
				items: [
					{ text: '侧边栏二级', link: '/blog/blog1' }, // 这里一级目录必须为/blog/
					{ text: '侧边栏二级', link: '/blog/blog2' }
				]
			}
		]
	},
	// 导航栏中其他网站的链接，icon可以是：discord，facebook，github，instagram，linkedin，slack，twitter，youtube
	socialLinks: [
		{ icon: 'github', link: 'https://github.com/vuejs/vitepress' },
		{ icon: 'twitter', link: 'https://www.baidu.com' },
	],
	// 首页底部设置footer
	footer: {
		message: 'Released under the MIT License.',
		copyright: 'Copyright © 2019-present Evan You'
	}
  }
})

6、全局组件 ​

1、创建docs/.vitepress/theme/index.js

javascript
import DefaultTheme from 'vitepress/theme'
import GlobalComponent from '../../../components/GlobalComponent.vue'
export default{
	...DefaultTheme,
	enhanceApp({app}){
		app.component('GlobalComponent',GlobalComponent)
	}	
}

10、github静态托管 ​

text
1、初始化本地仓库（生成.git文件）
git init
2、提交到暂存区
git add --all
3、提交到本地仓库
git commit -m "说明"
4、关联到github上的远程库
git remote set-url origin https://令牌@github.com/用户名/仓库名
5、提交到远程仓库
git push --set-upstream origin main
6、设置静态托管
Settings->Pages->Branch选择main-root（等30分钟，网络问题）-save
